1. Prerequisite: Python -- Python is a requirement for installing the Jupyter notebook.
2. Install Jupyter notebook and iPython kernel. The official website (http://jupyter.readthedocs.io/en/latest/install.html#installing-jupyter-using-anaconda-and-conda) contains more detailed information of the installation. The easiest way is install Anaconda.
3. To run the notebook, type "jupyter notebook"
4. Open the notebook from the notebook dashboard and click on Cell/Run all to run all the cells and view the results.
